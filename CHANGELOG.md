# Changelog / 更新记录

## 中文

### 0.1.0 - 2026-04-28

- 将项目整理为独立开源版本。
- 删除个人固定时间盒功能。
- 删除“AI 探索盒”功能。
- 为开源版添加独立本地存储身份，包括 IndexedDB 数据库名和 Electron session。
- 在设置页添加 JSON 导出、导入和清空本地数据能力。
- 添加 MIT License。
- 添加中英文 README、用户手册、贡献指南、本地存储文档、发布说明和 CI 工作流。

## English

### 0.1.0 - 2026-04-28

- Prepared the project as an independent open-source release.
- Removed the personal fixed timebox behavior.
- Removed the AI exploration box feature.
- Added independent local storage identity for the open-source app, including IndexedDB database name and Electron session.
- Added JSON export, import, and local data clearing in Settings.
- Added MIT License.
- Added bilingual Chinese/English README, user guides, contribution guide, local storage docs, release docs, and CI workflow.
